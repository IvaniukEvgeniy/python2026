# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Тренируем добавление, вставку и удаление элементов, чтобы управлять списком задач без создания новых копий.

## Условие задачи
Ты ведешь список дел по подготовке дачного домика к зиме. Нужно:
1. Создать список базовых работ.
2. Добавить в конец срочные пункты, пришедшие по SMS от соседа.
3. Вставить в середину напоминание о проверке сигнализации.
4. Удалить из списка выполненные задачи по индексу и по значению.
5. Переместить последнюю задачу на второе место с помощью комбинации `pop` и `insert`.
В конце выведи итоговый порядок дел.

## Пример входных данных
```python
works = ["закрыть окна", "слить воду", "убрать инструменты"]
urget = ["проверить крышу", "закупка соли"]
```

## Пример выходных данных
```
['закрыть окна', 'перенести генератор', 'слить воду', 'убрать инструменты', 'проверить крышу']
```

## Критерии успешного выполнения
- Применены минимум три разных метода изменения списка
- Есть вставка по индексу и удаление по значению
- Перестановка элементов выполнена без ручного обмена переменных
- Итоговый список распечатан после всех операций

## Подсказки
- Используй `extend`, чтобы добавить несколько задач сразу
- Вставку в середину можно сделать `insert(len(list)//2, "...")`
- Чтобы удалить элемент по значению, убедись, что он точно есть в списке
- Перемещение реализуй через `item = list.pop()` и `list.insert(1, item)`

## Частые ошибки и ограничения
- Не вызывай `remove`, если элемента нет — отлови это условие заранее
- Избегай одновременного перебора и модификации списка
- Не создавай новый список вместо работы с существующим
- Следи, чтобы индексы не выходили за длину списка при вставке
