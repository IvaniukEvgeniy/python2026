# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Нужно показать, что список помогает хранить упорядоченный набор вещей и быстро получать доступ к каждому элементу по индексу или через перебор.

## Условие задачи
Ты собираешь тревожный набор для короткой зимней поездки. Создай список обязательных предметов, добавь в него новые позиции от друзей, выведи количество элементов, покажи первый и последний предмет, а затем напечатай пронумерованный перечень для проверки перед выездом.

## Пример входных данных
```python
items = ["зарядка", "фонарик", "термос"]
extra = ["плед", "лопатка"]
```

## Пример выходных данных
```
Всего: 5
Первый: зарядка
Последний: лопатка
1. зарядка
2. фонарик
3. термос
4. плед
5. лопатка
```

## Критерии успешного выполнения
- Список создается в коде и дополняется без переопределения переменной
- Выводятся длина списка, первый и последний элементы
- Перебор реализован через цикл с нумерацией
- Пользователь видит обновленный список после всех операций

## Подсказки
- Начни с заранее заполненного списка, чтобы сразу было с чем работать
- Используй `len()` и индексы `0` и `-1`, чтобы не писать магические числа
- Для красивой нумерации подключи `enumerate` со значением `start=1`

## Частые ошибки и ограничения
- Не вставляй элементы через конкатенацию строк, используй методы списка
- Избегай ручного подсчета длины, чтобы не ошибиться при изменениях
- Следи, чтобы индекс `-1` применялся только к непустому списку
- Не удаляй существующие элементы, задача про добавление и вывод
