# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Учимся вырезать подсписки, работать с отрицательными индексами и шагом, чтобы быстро получать нужные данные без циклов.

## Условие задачи
У тебя есть список температур за неделю на даче. Нужно:
1. Получить последние четыре измерения.
2. Собрать каждый второй показатель, начиная со второго элемента.
3. Сделать срез, который возвращает значения только со среды по пятницу (ключевые середина недели).
4. Построить инвертированный список без `reversed`.
Выведи все результаты и объясни, чем отличается каждый срез.

## Пример входных данных
```python
temperatures = [1, -2, 0, 3, 5, 4, 2, -1, 0, 2]
```

## Пример выходных данных
```
Последние четыре: [4, 2, -1, 0, 2]
Каждый второй с индекса 1: [-2, 3, 4, -1]
Середина недели: [0, 3, 5]
Инверсия: [2, 0, -1, 2, 4, 5, 3, 0, -2, 1]
```

## Критерии успешного выполнения
- Используются только срезы, без ручных циклов
- Каждый пункт условия выводится отдельной строкой
- Инверсия построена через шаг `-1`
- Нет перепутанных границ или шага 0

## Подсказки
- Последние элементы получаются отрицательным стартом
- Чтобы взять каждый второй элемент, задай шаг `2`
- Средний диапазон удобно выбирать явными индексами `start:stop`
- Инвертировать список можно через `list[::-1]`

## Частые ошибки и ограничения
- Не путай порядок аргументов `start:stop:step`
- Следи, чтобы стоп-граница не включала лишние элементы
- Убедись, что шаг не равен нулю
- Не изменяй исходный список, работай только со срезами
