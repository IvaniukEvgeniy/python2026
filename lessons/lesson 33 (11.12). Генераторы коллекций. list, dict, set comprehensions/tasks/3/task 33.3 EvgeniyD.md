# Задание 33.3. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Dict comprehensions» показывает. как за одно выражение собрать словарь с нужными ключами и значениями без промежуточных циклов.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Есть список `employees`, в котором каждая запись — словарь с ключами `"name"`, `"hours"` и `"remote"` (True или False). Построй словарь `work_mode`, где ключом будет имя сотрудника, а значением строка `"удалённо"` или `"в офисе"`. При этом в словарь должны попасть только те сотрудники, у которых отработано не меньше 20 часов. Используй одно выражение dict comprehension.

## Пример входных данных
```
employees = [
    {"name": "Антон", "hours": 18, "remote": True},
    {"name": "Ирина", "hours": 24, "remote": False},
    {"name": "Марат", "hours": 32, "remote": True}
]
```

## Пример выходных данных
```
{'Ирина': 'в офисе', 'Марат': 'удалённо'}
```

## Критерии успешного выполнения
- Словарь `work_mode` создаётся одним dict comprehension.
- В него попадают только сотрудники с `hours >= 20`.
- Значение выбирается на основе поля `remote`.
- Исходный список `employees` не изменяется.

## Подсказки
- Проходи по списку так: `for emp in employees`.
- Внутри comprehension обращайся к полям через `emp["name"]` и так далее.
- Выбор строки сделай через тернарный оператор.

## Частые ошибки и ограничения
- Не добавляй внешний цикл и `dict()` вокруг comprehension.
- Не забывай фильтр по часам в конце выражения.
- Следи за тем. чтобы ключи словаря были уникальными.
