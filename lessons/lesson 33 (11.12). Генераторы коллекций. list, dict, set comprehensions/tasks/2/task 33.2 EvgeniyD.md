# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions. базовый рабочий инструмент» — тренируем превращение типичных циклов по спискам в одно выражение с фильтрацией.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Дан список пар `(название товара, остаток)`. Сформируй новый список `low_stock`, содержащий строки `"товар: N шт"` только для позиций, где остаток меньше 5, а название не пустое. Сделай это одним list comprehension.

## Пример входных данных
```
items = [
    ("Клавиатура", 12),
    ("Мышь", 3),
    ("Сумка", 0),
    ("", 9),
    ("Наушники", 4)
]
```

## Пример выходных данных
```
['Мышь: 3 шт', 'Сумка: 0 шт', 'Наушники: 4 шт']
```

## Критерии успешного выполнения
- Использован один list comprehension.
- Пустые названия пропускаются.
- В итог попадают только товары с остатком < 5.
- Формат строки совпадает с примером и использует текущий остаток.

## Подсказки
- Вспомни форму `[выражение for item in items if условие]`.
- Проверку пустого названия можно записать как `name.strip()`.

## Частые ошибки и ограничения
- Не добавляй `append` — результат должен создаваться comprehension-ом.
- Избегай вложенного цикла: достаточно одного прохода по списку.
- Не изменяй исходный список `items`.
