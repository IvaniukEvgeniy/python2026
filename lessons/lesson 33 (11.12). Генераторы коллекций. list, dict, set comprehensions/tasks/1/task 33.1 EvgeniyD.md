# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» — понимать, когда одно выражение может заменить цикл и сделать код компактнее без потери читаемости.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Тебе дали список дневных температур за неделю. Перепиши подготовительный цикл в одно выражение list comprehension. Нужно получить список подписей вида `"День 1: +15°C"` только для дней с температурой выше −5. Нумерация дней начинается с 1.

## Пример входных данных
```
temperatures = [-8, -2, 5, 11, 0, 3, -6]
```

## Пример выходных данных
```
['День 3: +5°C', 'День 4: +11°C', 'День 6: +3°C']
```

## Критерии успешного выполнения
- Используешь одно выражение list comprehension без дополнительного цикла.
- Температуры ниже или равные −5 не попадают в итоговый список.
- Формат строки строго как в примере, с номером дня и знаком температуры.
- Исходный список `temperatures` не изменяется.

## Подсказки
- Сперва вспомни синтаксис `[выражение for элемент in коллекция if условие]`.
- Используй `enumerate` и начни счёт с 1, чтобы получить номер дня.
- Для форматирования температуры можно добавить знак вручную через условие `"+" if value >= 0 else ""`.

## Частые ошибки и ограничения
- Добавление `print` внутри comprehension делает задачу громоздкой.
- Пропуск условия приводит к попаданию холодных дней в результат.
- Не изменяй исходный список температур; создавай новый список строк.
