# Задание 33.4. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Set comprehensions» помогает быстро получать уникальные значения и фильтровать данные без дополнительных циклов.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Дан список `logs`, где каждая запись — строка формата `"user:действие"`. Нужно построить множество `active_users`, содержащее только имена пользователей, у которых встречались действия `"login"` или `"update"`. Используй единственное выражение set comprehension и не меняй исходный список.

## Пример входных данных
```
logs = [
    "anna:login",
    "bob:logout",
    "anna:update",
    "claire:login",
    "bob:update"
]
```

## Пример выходных данных
```
{'anna', 'claire', 'bob'}
```

## Критерии успешного выполнения
- Множество формируется одним set comprehension.
- В результат попадают только пользователи с действиями `login` или `update`.
- Имя пользователя корректно отделяется от действия.
- Исходный список не модифицируется.

## Подсказки
- Разбей строку методом `split(":" , 1)` чтобы получить имя и действие.
- В условии проверяй принадлежность действия к набору допустимых значений.
- После comprehension выведи множество, чтобы убедиться в результате.

## Частые ошибки и ограничения
- Не добавляй `set()` вокруг comprehension.
- Следи, чтобы пустые строки не попадали в результат.
- В списках оставляй не больше пяти пунктов.
