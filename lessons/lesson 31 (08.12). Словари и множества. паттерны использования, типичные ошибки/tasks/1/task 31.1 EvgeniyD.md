# Задание 31.1. Словари и множества. паттерны использования, типичные ошибки
## Блок урока
Словарь как структура быстрого доступа — Блок показывает, как превращать повторяющиеся данные в словарь и мгновенно находить значения по ключу.

## Условие задачи
Сервис проката самокатов фиксирует, когда каждая станция проходила обслуживание. Оператор вводит один существующий и один неизвестный код, чтобы понять, нужен ли выезд инженера. Создай словарь repairs, где ключ — ID станции, а значение — дату последнего обслуживания. Затем выведи информацию для переменных requested_station и missing_station.

## Пример входных данных
```python
stations = [
    ('S1', '2025-12-01 10:30'),
    ('S8', '2025-12-02 09:10'),
    ('S3', '2025-12-03 18:40'),
]
requested_station = 'S8'
missing_station = 'S5'
```

## Пример выходных данных
```text
S8 -> 2025-12-02 09:10
S5 -> нет записи
```

## Критерии успешного выполнения
- Список пар преобразован в словарь repairs с ключом 'ID станции'.
- Для обращения к словарю используется метод get с значением по умолчанию.
- Выведены ответы для requested_station и missing_station в заданном формате.
- Код не падает при запросе отсутствующего ключа.

## Подсказки
- Пройдись по списку stations в цикле и наполняй словарь repairs.
- Используй множественное присваивание: key, value = запись из stations.
- Для отсутствующих данных возвращай строку 'нет записи'.

## Частые ошибки и ограничения
- Не обращайся к словарю через квадратные скобки, если нет проверки на существование ключа.
- Не оставляй ключи кортежами целиком — нужен именно идентификатор.
- Не дублируй вывод: достаточно одной строки на каждый запрос.
